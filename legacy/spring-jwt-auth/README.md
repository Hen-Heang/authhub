# spring-jwt-auth (legacy, archived)

Earlier, standalone JWT authentication practice project. Not part of the
AuthHub Gradle multi-project build (no entry in the root `settings.gradle`) —
kept here for reference only.

`security-api` in AuthHub supersedes this: same JWT-auth idea, plus OAuth2
login, OTP, password reset, and refresh tokens. Prefer `security-api` for new
work; this folder is not built, tested, or run as part of AuthHub.
